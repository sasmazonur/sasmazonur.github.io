  • FollowBot

    The FollowBot has been designed with the Intel RealSense Depth Camera D415 series that uses stereo vision to calculate the distance between robot and human. The human following method has the ability to differentiate a person and other objects. As well as follow a person in real-time using deep neural networks. The human following robot also has the ability to create 2D and 3D maps of its environment using simultaneous localization and mapping (SLAM). The 2D occupancy grid map is created from the laser scan data, and the mobile robot pose. And 3D map created by using the RGB-D Graph SLAM approach called Real-Time Appearance Based Mapping (RTAB-Map).

    The robot was implemented in Python, C++, C, C#, and has been tested in real-life testing scenarios.

    • Linear Regression & Logistic Regression(Python)  

      This programs implements linear regression and exploresome variations with it on this data. It uses the Boston Housing dataset of the housing prices in Boston suburbs. The goal is topredict the median value of housing of an area (in thousands) based on 13 attributes. Attributes describing the area(e.g., crime rate, accessibility etc)

    • Multinomial Naive Bayes for Sentiment Analysis(Python)  

      This program implements the Naive Bayes (Multinomial) classifier for Sentiment Analysis of an IMDB movie review dataset (a highly polarized dataset with 50000 movie reviews). The primary task is to classify the reviews into negative and positive. To transform from variable length reviews to fixed-length vectors, program uses the Bag Of Words technique. Program also provide pre-procession to remove html tags, remove special characters, convert text to lower case, replace punctuation characters with spaces and removing stopwords.

    • Random Forest & Decision Tree(Python)  

      The goal in this program is to develop variations of the decision tree algorithm and ensemble methods. The program classifies between Democratic and Re- publican majority vote in the 2016 US Election. Program do this based on a set of discrete features related to each county. These features are a categorical representation of continuous features, which are provided in the data description (county dictionary) file.

    • Unsupervised Learning for Human Activity Recognition(Python)  

      This program uses k-means clustering and Principal Component Analysis (PCA) methods to recognize human activity. The data comes from accelerometers and gyros of Samsung Galaxy S3 mobile phones. Each instance in the dataset is a set of feature values extracted from sensor signals and pre-processed to classify the type of activities a person has with their phone in his pockets. There are six activities WALK- ING, WALKING UPSTAIRS, WALKING DOWNSTAIRS, SITTING, STANDING, LAYING.
